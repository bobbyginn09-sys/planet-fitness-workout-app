# NEXSET 3.4.6 — Post-Workout Atlas Share

## New completion flow
- The existing workout-complete results screen now includes a dedicated **Coach Check-In Ready** panel.
- A prominent **Share with Atlas** button opens the iPhone native share sheet with the full NEXSET Coach Check-In report.
- A separate **Copy** button copies the same report for pasting directly into the current ChatGPT conversation.
- The report automatically includes the completed session, recent sessions, body trend, total statistics, next workout, and next-time recommendations.
- **Return home** remains available beneath the sharing controls.
