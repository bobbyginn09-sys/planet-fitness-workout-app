# Planet Fitness Workout Tracker App

This app is built for the 5 lifting day + 2 rest day Planet Fitness routine with no lunges.

## What it includes

- Fixed weekly split with 5 workouts and 2 rest days
- Start workout flow
- Set-by-set weight and rep logging
- Auto rest timer after sets
- Next-time weight recommendations
- Workout history
- Body weight/progress tracking
- Export/import backup
- Mobile-friendly design
- PWA files for Add to Home Screen/Install App when hosted over HTTPS

## How to use

1. Open `index.html` in a browser.
2. Choose today's workout or select another day.
3. Tap Start Workout.
4. Log each set's weight and reps.
5. Tap Finish + Save.
6. Check the recommendations next time you do that exercise.

## Backup note

Your data is saved in the browser on the device you use. Use Export Backup in the app before switching phones, clearing browser data, or moving to another browser.

## Install note

For the best app-like install experience, host this folder with GitHub Pages, Netlify, Vercel, or another HTTPS host, then open it on your phone and choose Add to Home Screen / Install App.
