# NEXSET 3.4.3 — Progressive Cardio & Coaching Update

## Active workout fit

- Rebalanced the weight and repetition controls on narrow iPhone screens.
- Reduced and constrained the weight-value typography so three-digit loads remain fully visible between the minus and plus buttons.
- Added tabular number spacing so weight values do not shift around while being adjusted.

## Cardio now progresses like a workout

Cardio is no longer logged as time alone. Scheduled cardio now captures:

- Duration
- Treadmill speed and incline, or stair-climber level
- Easy, Just right, or Hard effort
- Estimated treadmill distance
- A calculated next-session target that advances only one variable at a time

The active workout, previous-performance card, History, and Coach Check-In include the added cardio details. Progression raises duration first, then incline or stair level, then treadmill speed. A Hard result repeats the same target.

## Bike removed from the program

- Pull A now ends with **Stair Climber**.
- Legs A and Legs B now end with an **Incline Treadmill Walk**.
- Bike warm-up language was replaced with easy treadmill walking.
- No bike name or bike equipment remains in the patched scheduled routine.

## Workout corrections

### Legs A

- Smith Machine Squat: 4 × 8–10; repeat 130 lb cleanly before increasing.
- Romanian Deadlift: reduced from four to three working sets.
- 45° Leg Press: 3 × 12–15.
- Leg Extension: reduced from three to two working sets.
- Seated Leg Curl: reduced from three to two working sets.
- Calf Raise Machine: reduced from four to three sets, target 12–18.
- Incline Treadmill Walk: 12–20 minutes with speed and incline tracking.

**Total lifting workload: 17 sets.**

### Upper Definition

Reduced to 15 focused lifting sets:

- Machine Chest Press: 3 sets
- Neutral-Grip Lat Pulldown: 3 sets
- Seated Cable Row: 3 sets
- Machine Shoulder Press: 2 sets
- Rope Pushdown: 2 sets
- Cable Curl: 2 sets
- Controlled Incline Treadmill Walk: 10–15 minutes

Reverse Pec Deck and Dumbbell Lateral Raise were removed from this day to reduce overlap with Push A and Pull A.

## Stricter strength progression

NEXSET now raises weight only when every planned working set reaches the top of the rep range without a Hard rating. Mixed-weight sessions become a consistent working-weight target, incomplete sessions repeat, and hard-but-in-range sets hold the same load until control improves.

- Keeps data schema 5 and the existing NEXSET storage keys.
- Preserves workout history, body readings, templates, machine setups, backups, and unfinished workouts.
- Applies the revised program once to the saved editable routine.
- Uses the existing 3.4.2 Signature UI, icons, launch images, and anatomy assets.
