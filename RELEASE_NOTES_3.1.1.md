This release keeps the existing workout logic and saved-data format while bringing the interface much closer to the approved NEXSET visual direction.

- Compact NEXSET wordmark header and a short branded startup animation.
- Reference-inspired Home dashboard with a muscle-highlight workout card.
- Four quick actions: Body Weight, History, Planet Fitness, and Music.
- Compact progress summary for weekly consistency, total volume, and the next workout.
- Preserved flash-card active-workout logger with refined blue treatments.
- Program page redesigned as a seven-day schedule with recent workouts.
- Updated PWA cache, theme metadata, app icons, and launch screens.
- Existing local-storage keys and workout history remain compatible.
- Fixed bottom-sheet button handling so Close, Save, and quick-menu actions work reliably.
