# NEXSET 3.4.2 — Signature UI

NEXSET 3.4.2 is a complete visual refinement built on the full 3.4 flexible-workout feature set. It can be installed directly over an earlier NEXSET release; 3.4.1 does not need to be installed first.

## Signature home screen

- Rebuilt the Today experience around a single full-width training card.
- Added a deliberate anatomy spotlight instead of letting the figure float over the interface.
- Improved hierarchy for the workout name, target muscles, exercises, sets, duration, and starting exercise.
- Replaced the long stacked progress summary with three compact, glanceable insight cards.
- Reduced unnecessary height while preserving one-tap shortcuts and Quick Workout.

## Anatomical training maps

- Added new Retina-resolution `v342` anatomy assets for Push, Pull, Legs A, Legs B, Upper Body, and Recovery.
- Added restrained blue rim lighting and red target-muscle glow.
- Refined Legs A so quadriceps and calves read as anatomical regions instead of a full red leg covering.
- Refined Legs B so glutes, hamstrings, and calves are distinguished more clearly.
- Added controlled fading and framing so cropped upper-body figures look intentional.

## Workouts screen

- Replaced the long seven-card day list with a compact seven-day selector.
- Added one selected-day spotlight card with anatomy, focus, exercise count, set count, estimated time, start, and edit controls.
- Moved the full exercise list into an expandable Workout Outline.
- Kept Quick Workout, saved templates, recent sessions, routine editing, and rest-day actions accessible with less scrolling.

## Active workout

- Increased hierarchy around the current set and exercise name.
- Refined weight, rep, effort, warm-up, guidance, and bottom-action controls.
- Added a compact Next Exercise preview in unused vertical space.
- Improved progress contrast and set logging feedback.
- Preserved skipping, replacing, adding, reordering, equipment setup, notes, timers, set editing, and recovery.

## History, progress, and profile

- Rebuilt the empty History state into a proper onboarding screen.
- Added session, working-set, volume, and streak summaries when history exists.
- Unified Progress metrics, segmented controls, charts, and empty states with the new surface system.
- Refined Profile cards, navigation rows, quick-launch actions, and version presentation.

## Builders and sheets

- Improved Quick Workout and routine-editor sheet depth, spacing, search results, selected rows, and sticky actions.
- Preserved Quick Workout templates, program editing, exercise swaps, saved machine settings, backups, units, and themes.

## Compatibility

NEXSET 3.4.2 retains the existing storage keys and schema version. Existing workout history, body readings, settings, edited routines, templates, equipment notes, progression records, rest timers, and unfinished workouts remain compatible.
