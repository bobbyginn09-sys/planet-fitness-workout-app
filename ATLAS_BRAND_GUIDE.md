# NEXSET Brand Guide

## Brand position
NEXSET is a focused workout coach: direct, calm, useful, and data-driven. It avoids forced motivation and clutter.

## Name
- Product: **NEXSET**
- Descriptor: **Workout Coach**
- Tagline: **Progress starts with your next set.**

## Core colors
| Role | Hex | Usage |
|---|---|---|
| Primary Blue | `#2563FF` | Main actions, progress, active navigation |
| Primary Violet | `#7C3AED` | Gradient endpoint and brand highlights |
| Deep Navy | `#0E1117` | Primary app background |
| Slate | `#1A1F2B` | Cards, controls, secondary surfaces |
| Muted Gray | `#8B95A7` | Secondary text |
| Success | `#22C55E` | Completed sets and positive progress |
| Warning | `#F59E0B` | Caution and trend collection |
| Danger | `#EF4444` | Hard sets, destructive actions, errors |

## Type
Use the iPhone system typeface (`-apple-system`, SF Pro on iOS). Headings are bold and tightly tracked. Body text is regular with generous contrast.

## Voice
- Direct, not dramatic.
- Encouraging, not performative.
- Short, actionable coaching.
- Avoid “beast mode,” “grind,” “warrior,” and forced military language.

## UI principles
1. One screen, one job.
2. No scrolling during a workout.
3. Large one-thumb controls.
4. Previous performance and the next action are always visible.
5. Deeper data stays in Progress, History, or More.
