# NEXSET 3.2.0 — Release Hardening

- Added automatic local recovery snapshots while workouts and settings are saved.
- Added validated backup import, last-backup status, restore-local-copy, and diagnostics export.
- Added an in-app update prompt so new service-worker builds can install without manual cache rituals.
- Added duplicate active-workout protection and stronger active-session normalization.
- Added online/offline status messaging and runtime error metadata.
- Preserved `pfWorkoutApp.v1` and `pfWorkoutApp.active.v1` for existing workout history.
- Kept the established NEXSET 3.1.7 visual design unchanged.
